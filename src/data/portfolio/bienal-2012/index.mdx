{""}<a href="https://www.linkedin.com/company/grafikonstruct/" target="_blank">Grafikonstruct</a> hired me to design and implement a back-end solution comprised of a custom CMS and a RESTful JSON API to be consumed by a front-end mobile web application developed by their internal team.

I've built the client-side application of the CMS from scratch as a single-page app in <a href="https://angularjs.org" target="_blank">AngularJS</a>, resulting in a superior User Experience (for the time being). The back-end app was built in <a href="https://rubyonrails.org" target="_blank">Ruby on Rails</a> and deployed to <a href="https://www.engineyard.com" target="_blank">EngineYard</a>.

Some technologies used: HTML, CSS, JavaScript, AngularJS, Ruby on Rails, PostgreSQL, and EngineYard.
